rufus

The Reliable USB Formatting Utility (with Source)

Description

Rufus is a small utility that helps format and create DOS bootable USB flash drives, such as USB keys, USB memory sticks, etc.

It can be be especially useful for cases where:

  • you need to flash a BIOS or other firmware using a DOS utlity
  • you need to work on a system that doesn't have an OS installed
  • you want to run a specific DOS application

Despite its very small size (less than 100 KB), it provides you with everything you need.

System Requirements

Any version of Windows, from Windows XP. 32 or 64 bit doesn't matter.
If elevation is needed, you will be automatically prompted for it.

Usage

Download the executable and run it - no other installation is necessary!

Note that the executable is digitally signed, and was produced in a 100% transparent manner (see 'make release' in the source). Thus, as long as the signature states "Pete Batard - Open Source Developer", you can have confidence that it won't be trying to do anything malicious.
