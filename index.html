The Reliable USB Formatting Utility

Install

Download the exe and run it - no installation is necessary. You will be prompted for elevation on Windows platforms with UAC, as the program requires administrative access to be able to format USB drives. The downloadable executable from this site is signed, and generated exactly as described with 'make release' on MinGW32, so you can have some confidence that, even if it requires elevated privileges, it won't be trying to do anything malicious.
